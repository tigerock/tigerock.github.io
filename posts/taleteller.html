TaleTeller coming soon

March, 27 2024

How it comes

Months ago, as we start to create our visual novel like games. We found that the most important job is writing script. The tools for writing which is popular currently is YarnSpinner and Inkle. But they both have something we don’t agree.

Inkle

Ink is the script language that Inkle use. It maybe is the most popular script language which have 3.9K stars on GitHub. But we think it is like a programming script language than a stage script language.

There are lots of symbols in ink. A screenwriter who is lack of programming knowledge would be lost into it. And these symbols will influence the creator to clarify the context of the story .

Yarn

Yarn is the script language that YarnSpinner use. We like the grammar of Yarn. It is simple enough. But it still contains context related parts. Like its <<if>> directive. Also, when the nesting is too deep, it also makes the script less intuitive.

So we want to create a really simple tool to write these stories.

How the TaleTeller looks like

TaleTeller refers to the syntax of yarn. But removes context related directive.

What’s more, character & scenes management also are important parts of writing scripts. I mean, the scripts always contains dialog between characters, things happen in a specific scene. So TaleTeller contains characters and scenes management.

The goal is help screenwriter to write a complete story.

When would TaleTeller come out

Currently, it is under development. And we are not sure about some features are really needed or not. It would come out with a game.

And it would be multiple platforms supported including Windows, macOS and Ubuntu(We don’t have enough to test it on other linux variants distribution).
